## Installation

1. Download the project file.
2. Extract the zip file
3. Copy the folder and paste to htdocs.
4. Create database, name 'record_mngmnt_yasdo'.
5. Run the system to the browser.
6. It will process to create an account. (Account for admin.)
7. It will proceed to login page, enter account for admin.
